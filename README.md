# ADDCOIN

**The internet's homepage for fresh launches. A new coin every 10 seconds.**

ADDCOIN is what the web would look like if pump.fun shipped in 1999 — a pixel-faithful Windows 98 / Internet Explorer 6 / GeoCities throwback where the pop-up ads aren't hawking screensavers. Every ad is a brand new coin, launched live, right in front of you.

## The concept

Remember when the web was loud? Blinking banners, scrolling marquees, "You are visitor #00019362," a pop-up every time you blinked. ADDCOIN brings that energy back and aims it at crypto:

- **A coin launches every 10 seconds.** The ticker never stops.
- **Every pop-up is a launch.** Each ad advertises a real coin with live data — name, ticker, contract, market cap, momentum.
- **Latest Launches feed** so you never miss the next one.
- **Ad blocker toggle** — because we're not monsters. One click and the pop-ups go quiet. The launches keep coming.

## Features

- Faithful Win98 / IE6 / GeoCities interface, down to the title bar, toolbar and status bar
- Live launch feed: contract, market cap, 24h change and age
- Periodic pop-up "ads," each one a fresh coin
- One-click ad blocker with a live blocked counter
- Fast, static, and built to meme

## Tech

- Next.js (App Router) + React
- Hand-rolled retro CSS, zero UI dependencies
- Deployed on Vercel

## Run it locally

```bash
npm install
npm run dev
```

Then open http://localhost:3000.
